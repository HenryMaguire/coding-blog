Talking Machines with Henry

Hi, my name is Henry and welcome to my blog.

I'm a Physics PhD student in Manchester, UK who has got really interested in AI and Machine Learning over the last two years.

This blog is about taking Machine Learning ideas and making them into finished applications, showing all the development stages in between.

Neural Machine Translation

One of my projects is about Neural Machine Translation where I train a model to translate text from French to English.

In this project, I...

  • Build the translation system from scratch using Tensorflow
    • ...and, coming soon...
  • Make the model deeper, increasing translation accuracy by X%
  • Employ attention mechanisms and improve results by a further Y%
  • Embed the latest model in a web application using Flask

One Word Story

The second project is just a fun game idea that I had, called One Word Story. The user and a bot collaborate to create a story, taking turns to say one word at a time.

In this project, I...

  • Test a traditional NLP approach to story generation
    • ...and, coming soon...
  • Create a much more convincing model by using the power of Deep Learning
  • Put the game together and make a Flask Web application
  • Incorporate pre-trained speech recognition and text-to-speech models to improve the UX
